.dotfiles
========

Load and install "dotfiles"
----------------------

* Prerequisites
  * (apt-get) `stow`
*  Clone the repo into `~/.dotfiles` and run the setup tasks
  ```bash
  git clone git@github.com:zflat/.dotfiles.git ~/.dotfiles
  cd ~/.dotfiles && make all
  ```
